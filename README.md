#  Used Car Price Intelligence (Power BI + Machine Learning)

An end-to-end **Power BI analytics solution** that combines star-schema modelling, Python ML predictions, and interactive insights to help users understand **used car pricing, value, and market behaviour**.

The report integrates FACT/DIM tables, ML-generated predicted prices, AI visuals, and DAX measures to answer questions like:
- *Is this car overpriced?*
- *Which car segments give best value for money?*
- *How do brands, fuel types, and transmissions compare?*
- *What factors most influence resale price?*
